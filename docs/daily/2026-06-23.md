# AI Agent Daily Summary - 2026-06-23

## Key Updates

- **Deep Agents & Claude Code-like experiences**: Harrison Chase (@hwchase17) highlighted strong community work on building "your own Claude Code" using LangChain's Deep Agents. A detailed article explores the approach, noting particular strength with models like GLM-5.2.
- **Leve: Filesystem-first LangGraph agents**: New framework where agents are defined declaratively as a directory of files. Leve compiles the directory into a runnable, durable agent on LangGraph. Inspired by Vercel's Eve.
- **dcode harness**: Model-agnostic agent harness (Deep Agents Code) praised for production use. Works well outside tightly tuned environments like Claude Code. Example usage with Fireworks GLM-5p2.
- **"The Art of Loop Engineering" (LangChain)**: Major emphasis on moving beyond basic agent loops to a 4-layer architecture for reliability:
  1. Agent Loop (core tool-calling)
  2. Verification Loop (grader + rubric feedback)
  3. Event-Driven Loop (triggers via Fleet/LangSmith Deployment)
  4. **Hill Climbing Loop** — analysis agent reviews production traces and proposes improvements (self-improving systems)

## Hot Discussions

- Agents are becoming practical in real developer workflows. @steipete reported agents automatically triaging and implementing changes from GitHub issues in open source projects.
- Growing conversation around the difference between **frameworks** (LangGraph for stateful orchestration) and full **harnesses/runtimes** (Deep Agents, dcode) that provide opinionated, ready-to-run structures including planning, filesystem tools, and context management.
- Real-world embedded agents: @HamelHusain shared experience using an AI agent directly inside a bank UI.

## Actionable Insight

Prioritize building **Loop 2 (verification)** and **Loop 4 (hill climbing)** on top of LangGraph + LangSmith. The highest leverage for AI Native teams and individual "AI super individuals" is not just better prompts or models, but systematic self-improvement loops that learn from production traces. Start instrumenting with LangSmith traces today — they are the raw material for your hill-climbing agent.
